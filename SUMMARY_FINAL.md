## ğŸ†˜ TROUBLESHOOTING GUIDE

### Problem: Build Failed
```
Cause: Usually NPM dependency issue
Solution:
1. Check Railway logs
2. Usually non-blocking, app can still run
3. Continue to next step
Prevention: npm install sebelum push
```

### Problem: 500 Internal Server Error
```
Cause: Cache/config mismatch
Solution:
1. Open Railway shell
2. Run: php artisan config:cache
3. Run: php artisan cache:clear
4. Refresh browser
```

### Problem: Database Connection Error
```
Cause: MySQL not connected or credentials wrong
Solution:
1. Verify MySQL service added (Step 3)
2. Check database variables in Railway
3. Verify credentials match
4. Run migrations (Step 5)
```

### Problem: Gambar tidak muncul
```
Cause: Storage symlink not created
Solution:
1. Open Railway shell
2. Run: php artisan storage:link
3. Refresh browser
```

### Problem: Cannot find repository
```
Cause: Wrong repository name or not in GitHub
Solution:
1. Verify: 242410103011_Projek_Pweb exists
2. Verify: Logged in dengan nuhan-22
3. Check GitHub account: github.com/nuhan-22
```
